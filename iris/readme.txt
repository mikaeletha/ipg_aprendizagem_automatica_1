pre_processing.py
    Reads data from the iris dataset and creates a pre-processed dataset

histogram.py
    Given a dataset it produces an histogram for each variable
    python.exe .\histogram.py .\pre_processed\iris.csv

hist_dist.py
    Given a dataset it produces an histogram and distribution plots for each variable
    python.exe .\hist_dist.py .\pre_processed\iris.csv

outliers.py
    Shows box plot graphics (in order to check for outliers)
    python.exe .\outliers.py .\pre_processed\iris.csv
